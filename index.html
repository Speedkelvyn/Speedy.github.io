üíª Installation & Usage

Install Python

Download Python 3.8+ from python.org

Clone Repository

git clone https://github.com/Speedkelvyn/inboxzero.git
cd inboxzero

Install Dependencies

pip install -r requirements.txt

Configure Settings

Edit the script with your important senders and keywords

Example configuration:

IMPORTANT_SENDERS = [
'boss@company.com',
'@yourcompany.com'
]

IMPORTANT_KEYWORDS = [
'urgent', 'deadline', 'invoice'
]

Run the Script

python gmail_reader.py

Or for Outlook:

python outlook_reader.py
‚ö†Ô∏è First Run: The script will open your browser to authenticate. This is normal! Grant permissions and the script will save your credentials for future runs.
